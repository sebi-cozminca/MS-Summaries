# Exercise Summary

**Exercise 1: Access Lab Guide and Project Repository**
- **Goal:** Set up the lab environment and access the AI Red Teaming Agent project resources.
- **Actions:**
  - Log into GitHub and open the project repository at https://aka.ms/ignite25/lab516.
  - Create a Codespace and launch the interactive guide using mkdocs.
  - Keep the browser window open for step-by-step instructions.
- **Validation:**
  - Codespace is running, and the interactive guide is accessible in the browser.

**Exercise 2: Azure and VM Credentials Setup**
- **Goal:** Prepare Azure and Skillable VM credentials for use during the lab.
- **Actions:**
  - Verify and record assigned Azure subscription credentials (username, password, TAP token).
  - Note Skillable VM credentials for troubleshooting if needed.
- **Validation:**
  - Credentials are available and ready for use in setup and troubleshooting steps.

**Exercise 3: Run and Explore Red Teaming Agent in Codespace**
- **Goal:** Execute and interact with the AI Red Teaming Agent using Azure AI Foundry in a cloud-based development environment.
- **Actions:**
  - Follow the interactive guide to run automated red-teaming scans locally or in the cloud.
  - Use provided Azure credentials to authenticate and configure the environment as required.
  - Optionally revisit the lab at home using your own Azure subscription and the self-guided mode.
- **Validation:**
  - Red Teaming Agent runs successfully, and scan results are accessible; users can revisit and complete exercises independently.
