## Exercise Summary

**Exercise 1: Environment and Account Setup**
- **Goal:** Prepare the lab environment and ensure access to required resources.
- **Actions:**
  - Sign in to the provided virtual machine using lab credentials.
  - Create or sign in to a GitHub account as needed.
- **Validation:** Successful login to the VM and GitHub account.

**Exercise 2: Overview and Lab Orientation**
- **Goal:** Understand the scope and objectives of the lab.
- **Actions:**
  - Review the lab overview, objectives, and supported programming languages and frameworks.
  - Familiarize with the lab's focus on GitHub Copilot for paired programming and AI-driven coding assistance.
- **Validation:** User is aware of lab structure and expected outcomes.

**Exercise 3: Guided Lessons via GitHub**
- **Goal:** Complete hands-on lessons covering GitHub Copilot features and integrations.
- **Actions:**
  - Follow lesson links to GitHub for:
    - Getting Started with GitHub Copilot
    - Integrating MCP with Copilot
    - Using Advanced GitHub Copilot Features
    - Using Copilot for Azure to Deploy to the Cloud
    - Challenging Copilot with complex SQL
    - Upgrading Legacy Projects
    - Migrating to a New Language
  - Perform coding exercises, explore Copilot's autocomplete, inline chat, and slash commands.
  - Work with JavaScript, Python, C#, and other languages as guided.
- **Validation:** Each lesson's objectives are met, and successful code execution or feature demonstration is observed per GitHub instructions.

**Exercise 4: Lab Completion and Survey**
- **Goal:** Finalize the lab and provide feedback.
- **Actions:**
  - Complete the post-lab survey.
  - Submit responses and end the lab session.
- **Validation:** Survey is submitted and lab session is marked as complete.
