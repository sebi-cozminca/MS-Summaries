## Exercise Summary

**Lab Environment Access and Navigation**
- **Goal:** Enable users to access and navigate the AVS (Azure VMware Solution) demo labs for migration, optimization, and modernization scenarios.
- **Actions:**
  - Log into the provided virtual machine using supplied credentials.
  - Launch the AVS Demos via desktop shortcut or by navigating to https://aka.ms/AVSDemos in Edge.
  - Utilize on-screen tooltips, highlights, and keyboard shortcuts for guided navigation through the lab content.
  - Optionally toggle tooltips and highlights using provided keyboard shortcuts for a customized experience.
- **Validation:**
  - Successful access to the AVS Demos environment and ability to navigate through lab modules using shortcuts and UI guidance.

**Feedback and Completion**
- **Goal:** Collect user feedback and ensure lab completion is recorded.
- **Actions:**
  - Submit feedback via the provided link (https://aka.ms/ignite25-feedback).
  - Complete the lab and select "Submit" to record completion for credit.
- **Validation:**
  - Feedback is submitted and lab completion is reflected in the learning platform.
