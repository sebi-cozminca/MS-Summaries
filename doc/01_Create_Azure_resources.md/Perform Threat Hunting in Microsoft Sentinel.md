## Exercise Summary

**Exercise 1: Lab Setup and Environment Preparation**
- **Goal:** Prepare the Microsoft Sentinel lab environment and required resources.
- **Actions:**
  - Review lab disclaimers and prerequisites.
  - Download and configure necessary files and resources for the lab.
  - Access the Azure portal and provision required services.
- **Validation:**
  - Successful access to Azure portal and confirmation of resource availability.

**Exercise 2: Threat Hunting in Microsoft Sentinel**
- **Goal:** Perform threat hunting activities using Microsoft Sentinel's built-in tools and queries.
- **Actions:**
  - Access Microsoft Sentinel workspace in the Azure portal.
  - Use hunting queries to identify suspicious activities and potential threats.
  - Analyze results and document findings.
- **Validation:**
  - Identification of threats or confirmation of no suspicious activity; documented findings in the lab report.

**Exercise 3: Threat Hunting with Notebooks**
- **Goal:** Use Microsoft Sentinel notebooks for advanced threat hunting and data analysis.
- **Actions:**
  - Open and configure Jupyter notebooks within Microsoft Sentinel.
  - Run provided code cells to analyze security data and visualize results.
  - Interpret outputs and document insights.
- **Validation:**
  - Successful execution of notebook cells and meaningful analysis of security data.

**Exercise 4: Lab Completion and Submission**

- **Actions:**

  - Complete post-lab survey as required.
- **Validation:**
  - Confirmation of lab submission and receipt of completion credit in the learning system.
