# Exercise Summary

**Lab Overview: Unlock Your Agent's Potential with Model Context Protocol**
- **Goal:** Build a conversational AI agent for Zava, a retail DIY company, that analyzes sales data and helps customers find products using Microsoft Foundry Agent Service, Model Context Protocol (MCP), and PostgreSQL with advanced security and search features.
- **Actions:**
  - Log into the lab environment using provided credentials and access the lab guide.
  - Learn and apply Azure AI Foundry Agent Service to build and deploy AI agents with integrated tools and observability.
  - Use Model Context Protocol (MCP) to connect the Agent Service to external tools and data sources, enabling enhanced agent functionality.
  - Leverage Azure AI Foundry for rapid agent development and deployment.
  - Implement PostgreSQL with Row Level Security (RLS) and pgvector for role-based data protection and semantic search.
- **Validation:**
  - Agent is able to analyze sales data, answer product questions, and perform image-based product search.
  - Secure, role-based data access and semantic search are functional.
