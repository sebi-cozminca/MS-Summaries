# Exercise Summary

**Exercise 00: Set up the environment and import an agentic solution**
- **Goal:** Prepare the Power Platform environment and import a pre-built solution for agent development.
- **Actions:**
  - Configure Power Platform permissions and verify Dataverse is ready.
  - Import the Bookings solution into the environment and publish all customizations.
- **Validation:**
  - Solution is imported and customizations are published; environment is ready for use.

**Exercise 01: Explore the existing customer app**
- **Goal:** Understand the current model-driven app and customer scenario before adding AI enhancements.
- **Actions:**
  - Review the case study to identify business drivers and pain points.
  - Launch the Bookings model-driven app and create two sample property records.
- **Validation:**
  - Properties are created and visible in the app; data is stored in Dataverse.

**Exercise 02: Explore an agentic solution**
- **Goal:** Experience and analyze a Copilot agent that automates booking requests and integrates with Dataverse.
- **Actions:**
  - Test the Real Estate Copilot agent in Copilot Studio, tracking topic transitions and booking flow.
  - Review the Dataverse tables and schema supporting the agent's data operations.
- **Validation:**
  - Agent responds to prompts, books appointments, and interacts with Dataverse tables as expected.

**Exercise 03: Review how generative AI features complete the agentic solution**
- **Goal:** Enable and configure generative AI features to enhance agent responses and knowledge integration.
- **Actions:**
  - Enable generative AI in agent settings and configure the Conversational boosting topic.
  - Add public website and Dataverse table as knowledge sources.
  - Update the fallback topic to use generative answers and restrict to selected sources.
  - Configure authentication and security settings for the agent.
  - Test the agent's generative responses and knowledge integration.
- **Validation:**
  - Generative AI is enabled, knowledge sources are integrated, and agent provides context-aware, secure responses.

**Surveys**
