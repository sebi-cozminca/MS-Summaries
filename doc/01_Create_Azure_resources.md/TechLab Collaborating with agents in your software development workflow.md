# Tech Mastery Catalog Lab Instruction Summary

## Exercise Summary

**Exercise 1: Lab Environment Setup and GitHub Authentication**
- **Goal:** Prepare the lab environment and authenticate with the required GitHub organization.
- **Actions:**
  - Log in to the provided virtual machine.
  - Access the GitHub organization using supplied enterprise credentials.
  - Wait for account sync if necessary.
- **Validation:** Successful login to the VM and GitHub; access to the organization is confirmed.

**Exercise 2: Prerequisites and Environment Preparation**
- **Goal:** Set up all required tools and dependencies for agent-based software development workflows.
- **Actions:**
  - Follow instructions in the linked 0-prereqs.md GitHub document to install and configure prerequisites.
- **Validation:** All tools and dependencies are installed and verified as per the instructions.

**Exercise 3: Copilot Coding Agent Usage**
- **Goal:** Leverage GitHub Copilot as a coding agent to accelerate development tasks.
- **Actions:**
  - Follow the steps in 1-copilot-coding-agent.md to use Copilot for code generation and completion.
- **Validation:** Copilot is active and successfully assists with code authoring in the development environment.

**Exercise 4: Model Context Protocol (MCP) Integration**
- **Goal:** Integrate and utilize MCP for agent interoperability in the development workflow.
- **Actions:**
  - Complete the steps in 2-mcp.md to configure and test MCP integration.
- **Validation:** MCP-enabled tools are discoverable and usable by agents; interoperability is confirmed.

**Exercise 5: Custom Instructions for Agents**
- **Goal:** Enhance agent behavior and output by providing custom instructions.
- **Actions:**
  - Follow 3-custom-instructions.md to define and apply custom agent instructions.
- **Validation:** Agents respond according to custom instructions; output matches expectations.

**Exercise 6: Copilot Agent Mode in VS Code**
- **Goal:** Utilize Copilot's agent mode within Visual Studio Code for advanced collaboration.
- **Actions:**
  - Complete the steps in 4-copilot-agent-mode-vscode.md to enable and use agent mode features.
- **Validation:** Agent mode is enabled and collaborative features are functional in VS Code.

**Exercise 7: Reviewing Coding Agent Output**
- **Goal:** Review, validate, and iterate on code generated by Copilot and other agents.
- **Actions:**
  - Follow 5-reviewing-coding-agent.md to review, test, and refine agent-generated code.
- **Validation:** Code is reviewed, tested, and meets project requirements; feedback is incorporated as needed.
