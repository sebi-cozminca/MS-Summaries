### Exercise Summary

**Exercise 1: Sign into GitHub Enterprise and Get Started**
- **Goal:** Access and prepare the GitHub Enterprise environment for lab activities.
- **Actions:**
  - Log into the provided virtual machine using supplied credentials.
  - Access GitHub Enterprise at the specified SSO URL using the provided username and TAP code.
  - Navigate to the `skillable-events/ai-native-dev-lab` repository.
  - Launch a new Codespace on the main branch.
  - Authenticate and verify GitHub Copilot functionality in the Codespace.
  - Open and follow the instructions in the repository's README.md file.
- **Validation:**
  - Successful login to GitHub Enterprise and Codespace.
  - GitHub Copilot is authenticated and operational.
  - README.md instructions are followed and completed as directed.

**Exercise 2: Lab Completion and Submission**
- **Goal:** Complete the lab and ensure progress is recorded for credit.
- **Actions:**
  - Complete all required activities and surveys within the lab environment.
  - Select "Submit" in the lab interface to finalize completion.
- **Validation:**
  - Submission confirmation is received.
  - Lab completion status is updated in the learning system (visible the following week).
